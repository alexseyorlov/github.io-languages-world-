# Тренеровочный проект по веб-кодингу

Добро пожаловать! Этот репозиторий содержит тренеровочный проект для практики веб-кодинга. В нем вы будете работать с HTML, CSS и JavaScript, чтобы создать интерактивный отчет о глобальном распределении языков.

## Основные файлы

1. **index.html**: Структура страницы, включающая заголовок, навигацию, графики и таблицу данных.
2. **style.css**: Стили для оформления страницы.
3. **script.js**: Функциональность для обработки событий и управления элементами страницы.

## Инструкции по установке

1. Клонируйте репозиторий на ваш локальный компьютер:
   ```sh
   git clone https://github.com/your-repo-url.git
   ```

2. Перейдите в директорию проекта:
   ```sh
   cd your-project-directory
   ```

## Задачи

1. **Оптимизация навигации**:
   - Используйте атрибуты `data-*` для управления состоянием вкладок.
   - Сократите дублирование кода для переключения и сортировки.

2. **Добавление статистических данных**:
   - Обновите таблицу данных с актуальными данными о распределении языков.

3. **Инициализация графиков**:
   - Используйте Chart.js для отображения графиков вкладок "Распределение по континентам" и "Количество говорящих".

## Запуск проекта

1. Откройте файл `index.html` в веб-браузере.
2. Убедитесь, что все элементы корректно отображаются и функционируют.
